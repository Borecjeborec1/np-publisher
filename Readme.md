# np-publisher

np-publisher is simple library for pushing to github with 1 command 
```shell
  np-publisher
```


# Uses
```shell
  # For fixes
  np-publisher 
  # or
  np-publisher -fix

  # For minor changes
  np-publisher -minor
    # For major changes
  np-publisher -major
```

Or even shorter
```shell
  # For fixes
  np 
  # or
  np -fix

  # For minor changes
  np -minor
    # For major changes
  np -major
```
